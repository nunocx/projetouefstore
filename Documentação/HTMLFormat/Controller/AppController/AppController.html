beforeFilter

void beforeFilter( )

[line 47]

beforeFilter Esse método é responsável pelo filtro de autenticação que permite ou não um controller a efetuar qualquer método antes de executá-lo.

O componente Auth é o responsável por fazer essa verificação, autenticando um usuário para autorizar o método, ou não autorizando caso contrário. Nesse método são definidos parâmetros para as ações e redirecionamento de login e logout.
