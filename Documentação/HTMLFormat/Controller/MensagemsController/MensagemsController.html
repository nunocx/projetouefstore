pm_send_to Esse método envia uma mensagem de um usuário para outro.

O método busca o usuário destinatário no banco de dados através de um sql simples. Após o destinatário ser encontrado, a mensagem é ligada aos usuários emissor e destinatário, e caso seja referente a um produto, é ligada a este.
